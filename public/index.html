Connect in one command

One command issues your key and wires it into your IDE / Claude Code. Your traffic settles to your wallet — and earns USDC kickbacks while it runs.

# Issues an sk-shipyard key + points your IDE at this gateway
npx shipyard-inference connect --url https://YOUR-DEPLOYMENT

Or drop the key into any OpenAI SDK — the only change is base_url + your key:

from openai import OpenAI

client = OpenAI(
    base_url="https://YOUR-DEPLOYMENT/v1",
    api_key="sk-shipyard-…",  # your key from `connect`
)
resp = client.chat.completions.create(
    model="auto",  # let Shipyard pick the cheapest capable model
    messages=[{"role": "user", "content": "Hello from my agent"}],
)
print(resp.choices[0].message.content)
# Or curl it directly with your key
curl https://YOUR-DEPLOYMENT/v1/chat/completions \
  -H "authorization: Bearer sk-shipyard-…" \
  -H "content-type: application/json" \
  -d '{"model":"auto","messages":[{"role":"user","content":"hi"}]}'
